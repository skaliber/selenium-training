Bank Sample Application

Bank sample application is used to register user accounts and perform simple operations with these accounts.

User stories:

  • I as User want to register my account in the Bank to save money on it and to perform money operations
  • I as User want to perform money operation from my account to another one to pay/receive money to/from another User
  • I as User want to get money from my account
  • I as User want to put some money on my account to perform money operations

Acceptance criteria:

  • Each account has unique identifier
  • Each User can create some accounts
  • Account can't have negative amount of money on it
  • After money operation one of the accounts is increased and another is decreased by the same amount of money
  • Each money operation must have positive budget
  • User can't get more money from account than it has
  • User can't get zero or negative amount of money
  • User can't put zero or negative amount of money

Acceptance tests:

Available tests
Create some accounts by one user
Money operation
Negative Amount
