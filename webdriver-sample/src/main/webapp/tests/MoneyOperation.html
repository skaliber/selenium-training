Money operations

User can perform money operation with two different accounts to move positive amount of money from on to another. User can't move more money than account has.

Example:

John transfer to Bob 5 amount of money
open /bank/
waitForPageToLoad
clickAndWait link=Manage Accounts
type user Bob
type amount 10
clickAndWait //input[@value='Create']
type user John
type amount 7
clickAndWait //input[@value='Create']
clickAndWait link=Main Menu
clickAndWait link=Perform Operations
storeText //tbody/tr[1]/td[1] johnAccountId
storeText //tbody/tr[2]/td[1] bobAccountId
type fromAccountId ${johnAccountId}
type toAccountId ${bobAccountId}
type amount 4
clickAndWait //input[@value='Move']
assertText //tbody/tr[1]/td[3] 3.0
assertText //tbody/tr[2]/td[3] 14.0
