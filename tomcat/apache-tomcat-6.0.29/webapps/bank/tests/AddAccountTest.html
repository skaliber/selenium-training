Creation of some accounts by one user

Each user can create any number of accounts.

Example:

User Bob creates 3 accounts
open /bank/
waitForPageToLoad
clickAndWait link=Manage Accounts
type user Bob
type amount 3
clickAndWait //input[@value='Create']
assertText //tbody/tr[1]/td[2] Bob
assertText //tbody/tr[1]/td[3] 3.0
type user Bob
type amount 2
clickAndWait //input[@value='Create']
assertText //tbody/tr[1]/td[2] Bob
assertText //tbody/tr[1]/td[3] 2.0
type user Bob
type amount 0
clickAndWait //input[@value='Create']
assertText //tbody/tr[1]/td[2] Bob
assertText //tbody/tr[1]/td[3] 0.0
